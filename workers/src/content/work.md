# Selected work

## Patents

- **Orchestrator and interconnection fabric mapper for a virtual wireless base station**
  US12089071B2, United States. Granted.
  Massimo Notargiacomo, Jeffrey Courington, Stephen Turner, Jeffrey Masters, Vishal Agrawal, Francesco Foresta.
  Assignee: PPC Broadband Inc.
  A virtual wireless base station that can dynamically scale its capacity to meet changes in demand for connectivity. Includes virtual baseband modules, interface/router modules, an orchestrator module, and a fabric mapper module coupled by a low-latency switch fabric. The orchestrator determines current and near-future demand and instantiates or shuts down virtual baseband processors accordingly.

- **Integrated radio network with multi operator and multi signal format fronthaul capability**
  US11856449B2, United States. Granted.
  Massimo Notargiacomo, Gilberto Brizzi, Francesco Foresta, Alessandro Pagani, Giovanni Chiurco, Giulio Gabelli, Davide Durante, Fabrizio Marchese, Richard Wank, Michael Tierney.
  Assignee: PPC Broadband Inc.
  An integrated radio network hosting multiple network operators, each transmitting and receiving packetized prioritized streams over a fronthaul network. A switch/monitor identifies allocation violations and fronthaul bottlenecks, mitigating them by reducing or impeding low-priority packet streams.

- **System and method for securely hosting multiple network operators in a shared spectrum access system on a single virtual base station environment**
  US20210029549A1, United States. Published.
  Massimo Notargiacomo, Todd Landry, Jeffrey Masters, Roberto Orlandini, Jeffrey Courington, Francesco Foresta, Stephen Turner, Alessandro Pagani, Domenico Di Iorio, Kurt Jacobs, Patrick Henkle, Vishal Agrawal, Sasi Eswaravaka, Paul Stath.
  A virtual base station hosting multiple network operators and/or private networks in a single compute environment, with virtual baseband processors, a supervisor module, a fronthaul network interface, and a KPI coordinator module. Includes CBRS Daemons acting as proxy for obtaining grants to CBRS channels.

- **Blockchain-based method and system for securing a network of virtual wireless base stations**
  WO2021041937A1, WIPO (PCT). Published.
  Jeffrey Courington, Francesco Foresta, Vishal Agrawal.
  A system for securing a wireless telecommunications network capable of distributing licensed capacity via connection licenses. Local license servers coupled to virtual wireless base stations over a bus each implement a blockchain that logs transactions in which connection licenses change ownership among the virtual wireless base stations.

## Publications

### Peer-reviewed conference papers

- **Improving OpenStack Networking: Advantages and Performance of Native SDN Integration**
  IEEE International Conference on Communications (ICC), 2018.
  DOI: 10.1109/ICC.2018.8422753.
  Francesco Foresta, Walter Cerroni, Luca Foschini, Gianluca Davoli, Chiara Contoli, Antonio Corradi, Franco Callegati.
  **IEEE Best Paper Award — Communications Software, Services and Multimedia Applications Symposium (CSSMA).**
  Discusses OpenStack innovations enabling native SDN-like firewalling in the data plane and SDN-oriented control of virtual network infrastructure, with performance analysis at data and control/management plane levels toward production-level NFV deployments.

- **Implementation of Service Function Chaining Control Plane through OpenFlow**
  IEEE Conference on Network Function Virtualization and Software Defined Networks (NFV-SDN), 2017.
  DOI: 10.1109/NFV-SDN.2017.8169852.
  Gianluca Davoli, Walter Cerroni, Chiara Contoli, Francesco Foresta, Franco Callegati.
  Proof-of-concept implementation of the Service Function Chaining control plane combining OpenFlow with the IETF Network Service Header approach for flexible service function chain architectures.

- **Intent-based Management and Orchestration of Heterogeneous OpenFlow/IoT SDN Domains**
  IEEE Conference on Network Softwarization (NetSoft), 2017.
  DOI: 10.1109/NETSOFT.2017.8004109.
  Walter Cerroni, Chiara Buratti, Simone Cerboni, Gianluca Davoli, Chiara Contoli, Francesco Foresta, Franco Callegati, Roberto Verdone.
  Reference architecture and intent-based northbound interface for end-to-end service orchestration across multiple SDN/NFV technological domains, validated over a heterogeneous OpenFlow/IoT testbed.

- **Performance of Intent-based Virtualized Network Infrastructure Management**
  IEEE International Conference on Communications (ICC), 2017.
  DOI: 10.1109/ICC.2017.7997431.
  Franco Callegati, Walter Cerroni, Chiara Contoli, Francesco Foresta.
  Definition and proof-of-concept implementation of an intent-based northbound interface for orchestrating dynamic service chaining of virtualized network functions, with experimental evaluation of responsiveness under increasing load.
